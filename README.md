# student-management

*Student Management System – Spring Boot
Student Management System is a backend web application developed using Java Spring Boot that provides RESTful APIs to manage student data efficiently.
This project is designed to perform basic CRUD operations (Create, Read, Update, Delete) on student records and follows a clean layered architecture.
The application demonstrates the use of Spring Boot, Spring Data JPA, and Hibernate for building scalable and maintainable backend services. It is suitable for beginners to understand real-world backend development using Java.

*Technologies Used
Java
Spring Boot
Spring Data JPA
Hibernate
MySQL
Maven
REST API

*Features
Add new student details
Retrieve student details
Update existing student records
Delete student records
RESTful API design
Clean separation of Controller, Service, Repository, and Entity layers

*Project Structure
Controller – Handles HTTP requests
Service – Contains business logic
Repository – Handles database operations
Entity – Represents the database table
Main Application Class – Starts the Spring Boot application

*How to Run the Project
Clone the repository from GitHub
Open the project in Eclipse / IntelliJ
Configure MySQL database in application.properties
Run the StudentManagementApplication.java file
Test APIs using browser or Postman
